# 🏷️ Generador de Códigos de Barras y QR en C#

Este repositorio contiene una aplicación de **Windows Forms en C#** desarrollada en **Visual Studio 2022**, que permite **generar y guardar códigos de barras y códigos QR**.

---

## 📖 Funcionalidades

- ✅ Generar **códigos de barras (Code128)**.  
- ✅ Generar **códigos QR**.  
- ✅ Visualizar los códigos en pantalla.  
- ✅ Guardar los códigos generados en formato **JPEG**.  

---

## 🛠️ Requisitos

- **Visual Studio 2022** o superior.  
- **.NET Framework 4.x**.  
- Paquete NuGet: [`Zen Barcode Rendering Framework`](https://www.nuget.org/packages/Zen.Barcode.Rendering.Framework/).  

Instalación desde la consola de NuGet:

```powershell
Install-Package Zen.Barcode.Rendering.Framework
```

---

## 📂 Archivos principales

- `Código de Barras y QR.sln` → Solución principal.  
- `Código de Barras y QR.csproj` → Proyecto de Windows Forms.  
- `Form1.cs`, `Form1.Designer.cs`, `Form1.resx` → Formularios principales.  
- `Program.cs` → Punto de entrada de la aplicación.  
- `Properties/` → Configuraciones del proyecto.  
- `App.config` → Configuración inicial.  
- `packages.config` → Dependencias NuGet.  

📦 Además, este repositorio incluye el archivo `Código de Barras y QR.zip`.  
Puedes descomprimirlo y abrir directamente el archivo **`.sln`** en Visual Studio.  

---

## 🚀 Ejecución

1. Clona el repositorio:  
   ```bash
   git clone https://github.com/alangaquinor/codigo-barras-qr.git
   ```
2. Abre la solución **`Código de Barras y QR.sln`** en Visual Studio.  
3. Restaura los paquetes NuGet.  
4. Ejecuta el proyecto con **F5**.
